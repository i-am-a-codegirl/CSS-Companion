# Css Companion
The only CSS compoenent library you need for creating customizable, compatible and beautiful component for your websites.

 <h3><a href="https://the-css-companion.netlify.app/index.html">Check out CSS Companion</a></h3>

 # Installation

- You can add this line into your html code to use the components inside the library.
```html
<link rel="stylesheet" href="https://the-css-companion.netlify.app/component/component-css.css" />
```
- You can also import it in your css file.
```css
@import url("https://the-css-companion.netlify.app/component/component-css.css");

```

# Key Features
▶️ The website and all the components are responsive. <br/>
▶️ All the components \are customizable and can be easily deployed.

# Snack-it

An E-commerce website / store to buy healthy snacks online from across the globe. Build using HTML, CSS, & Javascript!
 <h3><a href="https://the-css-companion.netlify.app">Check out CSS Companion</a></h3>


# Key Features
▶️ All the pages and components are responsive.
▶️ All the components support both dark and light theme
▶️ Home Page
▶️ Product Listing Page
▶️ Filters by Price range slider, category, sort by, category, and rating range slider
▶️ Cart Management
▶️ Wishlist Management
▶️ Authentication (Signup, Login, and Logout)
